System Requirements:
JDK (Java Development Kit) - JDK must be installed on the system.

Procedure

1. Go to the GitHub link and click on code .
2. Download the Zip file from there.
3. Extract all the files in your system. 
4. Window Users double click on the batch file and command prompt will appear. 
5. Then, enter the absolute path of your pdf file that you want to split. 
6. Enter the absolute path where you want to store your split pdf files. 
7. Enter the keyword on the basis of which you want to split your pdf. 
8. Then a folder with the name ‘Split’ will be created and it will contain all your split
    pdfs. 
9. If you are running the code again, you need to give a new path or remove the
   ‘Split’ folder from that location.
